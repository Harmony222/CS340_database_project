Novel Hovel Index Page

The Novel Hovel is a quaint local bookstore with a long-standing reputation for hosting the most organized and sensational (in-person!) book clubs. Then COVID happened. In order for the Novel Hovel to continue to serve its customers, the company is in need of a database driven website to reestablish a connection with its community. Novel Hovel organizes and hosts an average of 8 book club meetings per month and engages more than 300 community members.

Novel Hovelâ€™s needs include the ability to:
  • Maintain a record of members and member contact information
  • Have a system to oversee all the different book clubs managed by Novel Hovel including their theme, books, meeting frequency, etc
  • Keep track of which members are part of which book club(s)
  • Know when a book club meeting is happening and which members are signed up to attend
  • Allow members to sign up for book clubs and meetings

Outline

  • Members
    • View all members - SELECT FROM Members
    • Become a member - INSERT INTO Members
  • Book Clubs
    • View all book clubs - SELECT FROM BookClubs
      • View members for specific book club - SELECT FROM bookclubs_members
    • Create a new book club - INSERT INTO BookClubs
    • Sign up for a book club - INSERT INTO bookclubs_members (M:M relationship)
  • Meetings
    • View all future meetings - SELECT FROM Meetings
      • UPDATE ClubMeetings
      • DELETE FROM ClubMeetings
    • Schedule a new meeting - INSERT INTO Meetings
      • Dynamic drop down menu of books filtered by genre, also only includes those books not already assigned to a meeting - SELECT FROM Books
    • Sign up for a meeting - INSERT INTO meetings_members (M:M relationship)
      • Club meetings list dynamically created from book club dropdown menu - SELECT FROM ClubMeetings
    • View members attending a meeting - SELECT FROM meetings_members
      • Remove member from a meeting - DELETE FROM meetings_members (M:M relationship delete)
      • Club meetings list dynamically created from book club dropdown menu - SELECT FROM ClubMeetings
  • Books
    • View all books - SELECT FROM Books
      • DELETE FROM Books - if a book selected for a ClubMeeting is deleted, meetingBookID will be "NULL"
    • Add a book - INSERT INTO Books
  • Genres
    • View all genres - SELECT FROM Genres
    • Add a genre - INSERT INTO Genres
